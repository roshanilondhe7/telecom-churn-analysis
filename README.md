# 📊 Telecom Customer Churn Analysis

## Project Overview
Exploratory Data Analysis (EDA) on a 
telecommunications dataset of 4,250 customers 
to identify churn drivers and build a customer 
risk profile using Microsoft Excel.

## 🎯 Key Findings
- Overall churn rate: **14.07%** 
  (below industry average of 15–25%)
- **86%** of churned customers had 
  no voicemail plan
- Churned customers average **30 years old** 
  vs 40 for loyal customers
- Churned customers pay **$5.69 more per day**
- **NJ** has highest churn rate at 27.1%
- Voicemail plan = strongest retention factor

## 👤 High Churn Risk Customer Profile
| Factor | High Risk Indicator |
|--------|-------------------|
| Age | Under 35 years old |
| Voicemail | No voicemail plan |
| Daily Usage | Above 200 mins/day |
| Daily Charge | Above $35/day |
| CS Calls | 3 or more calls |
| Location | NJ, CA, WA, MD, MT |

## 💼 Business Recommendations
1. **Promote voicemail plans** — free trials 
   for customers without one, especially under 35
2. **Create high-usage pricing plans** — 
   discounts for 200+ mins/day users
3. **Improve customer service** — proactively 
   contact customers making 3+ support calls
4. **Target young customers** — loyalty 
   programmes for under-35s
5. **Focus on high-risk states** — regional 
   campaigns for NJ, CA, WA, MD and MT
